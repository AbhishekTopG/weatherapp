# weatherapp
## INSTALLATION:
run ``pip install -r requirements.txt``

make a file ``SECRETS.py``

enter ``x = 'API KEY'`` in the ``SECRETS.py`` file

run ``run.py``


The client endpoint is avaliable at http://127.0.0.1:5000


**Dont forget to update your openweather api key**
